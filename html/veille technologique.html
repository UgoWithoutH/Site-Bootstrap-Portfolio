La mise à jour KB5003214 de Windows 10 que Microsoft vient tout juste de déployer provoque des soucis dans la barre des tâches : certaines icônes ne s'affichent plus ou se chevauchent, rendant difficile le lancement d'applications ou de raccourcis vers les modules de Windows, notamment celui lié à la gestion du réseau.
Windows 10 21H1 n'apporte que très peu de nouveautés fonctionnelles, Microsoft s'étant concentré sur la correction de bugs et les optimisations. Il faudra sans doute attendre la finalisation du projet Sun Valley qui arrivera peut-être avec la version 21H2, à l'automne 2021, pour profiter de réelles améliorations dans Windows 10, notamment dans l'interface.
Microsoft travaille sur une mise à jour d'envergure pour Windows. Connue sous le nom de code Sun Valley, elle devrait apporter des changements profonds pour revitaliser l'OS de Microsoft.
Plusieurs problèmes de sécurité ont été découverts dans Thunderbird, qui pourraient avoir pour conséquence l'exécution de code arbitraire.
Plusieurs vulnérabilités ont été découvertes dans le navigateur web Chromium.
Plusieurs problèmes de sécurité ont été découverts dans le système de base de données PostgreSQL qui pourraient avoir pour conséquence l'exécution de code arbitraire ou la divulgation du contenu de la mémoire.
